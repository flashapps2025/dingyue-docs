# 邮箱配置指南

概述 本指南将帮助您配置开发者账户的邮箱连接，用于接收 Apple 开发者相关的邮件通知。系统支持所有使用 IMAP 协议的邮箱服务。

## 支持的邮箱服务

- Gmail
- Outlook/Hotmail
- Yahoo Mail
- iCloud Mail
- 163.com
- 139.com
- QQ.com
- 企业邮箱（支持 IMAP 协议）
- 其他支持 IMAP 的邮箱服务

## 配置步骤

### 1. 基本信息配置

#### 邮箱别名（可选）

- **说明**：用于在系统中显示邮箱的友好名称
- **示例**：`Apple 开发者邮箱`、`主邮箱`等
- **注意**：此字段为可选，不填写将使用邮箱地址作为显示名称

#### 服务器地址（必填）

- **说明**：邮箱服务器的 IMAP 地址
- **常见配置**：
  - Gmail: `imap.gmail.com`
  - Outlook: `outlook.office365.com`
  - Yahoo: `imap.mail.yahoo.com`
  - iCloud: `imap.mail.me.com`
  - 163.com: `imap.163.com`
  - 139.com: `imap.139.com`
  - QQ.com: `imap.qq.com`

#### 服务器端口（必填）

- **说明**：IMAP 服务器的端口号
- **常见配置**：
  - SSL/TLS: `993`
  - 非加密: `143`（不推荐）

#### 用户名（必填）

- **说明**：完整的邮箱地址
- **示例**：`your-email@gmail.com`
- **注意**：某些企业邮箱可能需要使用完整的邮箱地址

#### 密码（必填）

- **说明**：邮箱密码或应用专用密码
- **注意**：对于启用了两步验证的邮箱，需要使用应用专用密码

### 2. 常见邮箱配置示例

#### Gmail 配置

```
服务器地址: imap.gmail.com
服务器端口: 993
用户名: your-email@gmail.com
密码: 应用专用密码（16位）
```

**获取 Gmail 应用专用密码：**

- 登录 Google 账户
- 进入"安全性"设置
- 启用"两步验证"
- 生成"应用专用密码"
- 使用生成的16位密码

**详细说明：**[Google 应用专用密码帮助文档](https://support.google.com/accounts/answer/185833?hl=zh-Hans)

#### Outlook 配置

```
服务器地址: outlook.office365.com
服务器端口: 993
用户名: your-email@outlook.com
密码: 邮箱密码或应用密码
```

**Outlook 应用密码设置：**

- 登录 Microsoft 账户
- 进入"安全性"设置
- 启用"两步验证"
- 创建"应用密码"
- 使用生成的应用密码

#### 163.com 配置

```
服务器地址: imap.163.com
服务器端口: 993
用户名: your-email@163.com
密码: 邮箱密码或客户端授权密码
```

**163.com 客户端授权密码设置：**

- 登录 163 邮箱
- 进入"设置" → "POP3/SMTP/IMAP"
- 开启"IMAP/SMTP服务"
- 生成"客户端授权密码"
- 使用生成的授权密码

#### 139.com 配置

```
服务器地址: imap.139.com
服务器端口: 993
用户名: your-email@139.com
密码: 邮箱密码或客户端授权密码
```

**139.com 客户端授权密码设置：**

- 登录 139 邮箱
- 进入"设置" → "POP3/SMTP/IMAP"
- 开启"IMAP/SMTP服务"
- 生成"客户端授权密码"
- 使用生成的授权密码

#### QQ.com 配置

```
服务器地址: imap.qq.com
服务器端口: 993
用户名: your-email@qq.com
密码: 邮箱密码或授权码
```

**QQ.com 授权码设置：**

- 登录 QQ 邮箱
- 进入"设置" → "账户"
- 开启"IMAP/SMTP服务"
- 生成"授权码"
- 使用生成的授权码

#### Yahoo Mail 配置

```
服务器地址: imap.mail.yahoo.com
服务器端口: 993
用户名: your-email@yahoo.com
密码: 应用专用密码
```

**Yahoo 应用专用密码设置：**

- 登录 Yahoo 账户
- 进入"账户安全"设置
- 启用"两步验证"
- 生成"应用专用密码"
- 使用生成的专用密码

### 3. 测试配置

配置完成后，点击"测试配置"按钮验证连接：

- ✅**成功**：显示"邮箱配置测试成功！"
- ❌**失败**：显示具体错误信息，请检查配置

### 4. 常见问题

#### 连接失败

- 检查网络连接
- 确认服务器地址和端口正确
- 验证用户名和密码
- 检查邮箱是否启用了 IMAP

#### 认证失败

- 确认密码正确
- 对于 Gmail/Yahoo，使用应用专用密码
- 对于 163/139/QQ，使用客户端授权密码
- 检查是否启用了两步验证

#### 端口问题

- 确认使用正确的端口号（通常为 993）
- 检查防火墙设置
- 尝试使用不同的端口

#### 国内邮箱特殊说明

- **163.com/139.com/QQ.com**：需要开启 IMAP 服务并生成客户端授权密码
- **授权密码**：不是邮箱登录密码，需要在邮箱设置中单独生成
- **安全考虑**：建议定期更换授权密码

### 5. 安全建议

- **使用应用专用密码**：对于支持的应用，使用专用密码而非主密码
- **定期更新密码**：定期更换邮箱密码和授权密码
- **启用两步验证**：为邮箱账户启用两步验证
- **网络安全**：确保在安全的网络环境下进行配置
- **授权密码管理**：妥善保管客户端授权密码，不要泄露给他人
