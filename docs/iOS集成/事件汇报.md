# 事件汇报

订阅内购页中自行汇报相关事件

当需要收集事件时，调用事件汇报方法：

```swift
import DingYue_iOS_SDK
DYMobileSDK.track(event: "CLICK_BUTTON", extra: "test", user: "test") // event为必传项, extra和user为可选项
```

```objective-c
@import DingYue_iOS_SDK;
[DYMobileSDK trackWithEvent:@"CLICK_BUTTON" extra:nil user:nil]; // event为必传项, extra和user为可选项
```
