# 第三方归因汇报

如果您已集成了 Adjust、AppsFlyer、Amplitude，则可报告归因数据。

<CodeGroup>

<CodeGroupItem title="swift">

```swift
DYMobileSDK.reportAttribution(adjustId: String?, appsFlyerId: String?, amplitudeId: String?)
```

</CodeGroupItem>

<CodeGroupItem title="objective-c">

```objective-c
[DYMobileSDK reportAttributionWithAdjustId:adjustId appsFlyerId:appsFlyerId amplitudeId:amplitudeId];
```

</CodeGroupItem>

</CodeGroup>
