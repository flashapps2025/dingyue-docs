# 内购页

使用Dingyue管理您的内购页，不需要通过苹果审核，制作完毕即可下发到用户群体，我们提供数十种模板供您选择，力求帮您提升付费率，当然，您也可以选择上传自制的内购页。

## 如何在线制作一个内购页？

选择添加按钮后，选择系统模板或者自定义内购页，系统模板是我们免费提供的模板；自定义内购页需要您上传自制内购页，下载内购页模板后，修改模板内的素材后上传，依旧可以编辑价格等元素。

![模板编辑](/images/11.png)

以系统模板为例，选择您喜爱的模板样式后，开始编辑。

![模板编辑](/images/12.png)

制作内购页时，根据您的需求填写右侧项目，您可以改变素材、按钮、颜色等视觉元素。订阅组需要绑定创建好的订阅项。 您可以在菜单栏找到「订阅配置」来创建，或者直接点击“创建订阅项”。除此之外，您也可以为内购页创建多语言，选择添加语种为不同区域的用户展示本土化的内购页。

制作完成后，由于您还没有创建用户群体，所以只能发布给ios默认用户组，默认组包含全体用户，如果您需要下发给特定的用户群，则需要在「用户管理」创建。您可以先选择保存，随后创建好用户群后和内购页进行绑定。

## 自定义内购页

![模板编辑](/images/13.png)

自定义内购页是由h5实现，工程师可以下载内购页模板并在给定模板的基础上自行更改页面样式，并按模板内文档要求配置订阅项，然后将写好的文件压缩成zip包上传至网站即可创建成功自定义内购页。

自定义内购页在页面布局上更为灵活，同样是通过服务器下发方式去 展示。

## 本地内购页

![模板编辑](/images/14.png)

本地内购页同样是通过h5实现，书写方式同自定义内购页。创建时仅需配置订阅项，上传预览图（此预览图仅用于区分不同页面，可不传）等即可创建成功，创建成功后需要将自己写好的内购页包名更改为自动生成的页面id，放入主工程中。后续由工程师设置本地内购页路径后即可正常显示。

本地内购页可以实现更快的加载速度，可一定程度避免由于网速导致页面加载不成功的情况。
