# 订阅配置

订阅配置目的是填加申请过的所有订阅价格，不再局限于App Store的订阅组，您可以自由组合所有订阅价格，并分配到内购页上。

需要注意的是，配置的所有价格都是您在App Store Connect申请过的价格，我们不能凭空帮您创造新的订阅价格。

选择添加按钮后，填写弹框里的项目完成配置。

![添加订阅项](/images/21.png)

## 订阅项名称

自定义的名称，区分不同的订阅项，例如：月订阅1

## 订阅类型

在下拉菜单中，选择匹配的订阅类型

![选择订阅类型](/images/22.png)

## 订阅产品ID

每添加一个订阅价格，您都需要填写相对应的订阅产品ID。

登陆App Store Connect，选择应用程序后在「功能」 >「订阅」中选择订阅价格所在的订阅群组。

![订阅产品ID位置](/images/23.png)

以月订阅为例，找到对应的产品ID。


## 币种 & 订阅价格

选择您作为换算基准的币种，并填写在App Store Connect订阅中相对应的价格，我们会帮助您换算成其他国家的对应价格，换算基准与苹果官方保持一致，不必担心会对消费者产生价格歧义。

订阅价格配置完成后，您就可以在导航栏找到「内购页」，制作内购页并且下发价格了。
