# 为什么选择Dingyue

Dingyue有效帮助运营和相关开发人员分析并增加订阅量，您可以用来维护老用户，拓宽新用户，甚至可以进行App Store广告投放。

## 内购页管理

- 海量内购页模板在线制作
- 自由组织申请的内购价格
- 划分用户群体，配置量身定做的内购页

## 营销和测试

- 使用促销活动为用户提供个人优惠，进行购买和查看订阅状态。
- 订阅计划的A/B 测试。为您的订阅测试不同的人群、价格、持续时间、试用期以及不同的视觉元素。
- 订阅分析让收益一目了然，并收集数据可供分析。
- 查看用户的订阅周期，为用户提供不同类型促销订阅。
- 集成。支持AppsFlyer、Adjust、Mixpanel、Facebook、Amplitude和阿里云、amazon的第三方归因和分析

## 广告优化

专为国人使用习惯设计的App Store自助广告投放服务，您可以在这里添加您的应用程序并搭建计划，修改、删除、添加广告系列，下载关键词报告等。不需要切换Apple后台，一次配置终身享受，让投放简单化。
