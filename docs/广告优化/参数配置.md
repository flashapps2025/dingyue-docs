# 参数配置

如果您需要使用订阅平台便捷的添加苹果ASA推荐词，请配置以下内容。

![选择对象](/images/67.png)

## Search Ads后台

1.登录苹果Search Ads后台，<https://app.searchads.apple.com/cm/app/report> ；任意选中一个在投的广告系列点击进入。

![选择对象](/images/68.png)

2.进入后先按F12进入开发者模式；点击network查看所有网络请求；filter框中输入recommendation过滤得到我们想要的结果；最后点击添加广告组。

![选择对象](/images/69.png)

3.进入添加广告组页面后苹果会请求获取推荐词的接口，这时我们会在network过滤条件中看到此接口，右键复制选择Copy as cURL(cmd)。

![选择对象](/images/70.png)

4.复制成功后打开文本编辑软件粘贴进去，然后就可以在里面找到我们需要的信息了。

分别在文本中搜索sa_user和XSRF-TOKEN-CM找到对应的内容复制到订阅网站指定位置中。

![选择对象](/images/71.png)

5.点击保存即可设置成功。注：此信息一段时间后token会失效，苹果Search Ads后台重新登录后请重新按照上述步骤更新此信息。

![选择对象](/images/72.png)

## 查看数据

设置成功后在给该广告账户的广告组添加关键词时，就能便捷的把苹果推荐词添加至广告组中。
