# 用户管理

利用过滤器配置用户群，用户群可以进行A/B测试，也可以在「内购页」绑定制作好的内购页，避免更新app，帮您改变用户的购买价格。

## 创建用户群

ios默认用户组包含全部用户，您可以建立用户群来划分用户。

![用户群入口](/images/29.png)

添加过滤器，可以根据系统版本、设备型号、激活时间、地区等数十项进行筛选。

![过滤器示例](/images/30.png)

添加完毕过滤器后，选择“下一步”，为您的用户群创建名称，也可根据需求添加“开关”。开关用于实现服务器的某些特定功能，我们会帮您返回您自定义的开关字段，例如您的应用程序代码中使用Ads Switch来控制广告的展示与否，那么您可以在Dingyue添加Ads Switch的开关，打开开关，则展示广告；关闭开关，则不展示广告。

简而言之，“开关”是帮助您便捷管理应用程序功能的服务，您可以利用开关操作应用程序内的某些功能，而非修改代码。

![开关示例](/images/31.png)

创建好用户群后，就可以绑定内购页或者进行A/B测试！
